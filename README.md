# AI-Powered PC Builder

## Overview
This project is an **AI-powered PC builder application** designed to help users build the best possible personal computer based solely on their requirements, even if they have little or no knowledge of PC components. The system leverages the **Gemini API** to provide intelligent build recommendations and supports both **guided** and **custom PC building** workflows.

In addition, the platform includes a **global component search feature** that aggregates product listings from multiple online sources, identifies the lowest available prices, and presents direct purchase links to users.

---

## Key Objectives
- Enable non-technical users to build optimal PCs using requirement-based input.
- Provide AI-assisted component selection using Gemini API.
- Ensure full hardware compatibility in custom PC builds.
- Help users find the lowest-priced components across multiple vendors.
- Offer a clean, responsive, and user-friendly interface.

---

## Core Features

### 1. AI-Based PC Builder (Gemini API)
- Users specify requirements such as:
  - Budget
  - Primary use (gaming, productivity, programming, content creation, etc.)
  - Performance preferences
- The Gemini-powered builder automatically:
  - Selects compatible components
  - Balances performance and cost
  - Generates a complete PC build recommendation

---

### 2. Custom PC Builder with Compatibility Handling
- Advanced users can manually select individual components.
- The system validates compatibility in real time, including:
  - CPU and motherboard socket matching
  - RAM type and frequency support
  - GPU and case clearance
  - PSU wattage requirements
- Incompatible selections are flagged instantly with corrective suggestions.

---

### 3. Global Component Search and Price Comparison
- Searches PC components across multiple e-commerce platforms.
- Aggregates results into a unified list.
- Automatically:
  - Sorts components by lowest price
  - Displays vendor information
  - Provides direct links to product pages
- Helps users make cost-effective purchasing decisions.

---

### 4. User-Friendly Interface
- Clean and responsive frontend built with React.js and CSS.
- Simple requirement-based workflow for beginners.
- Detailed build summaries for transparency and learning.
